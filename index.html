Welcome to DebAIt

DebAIt is a simple web app where you can debate with an AI, explore arguments on both sides of an issue, and see what other users think.

Use the navigation above to explore:

  • Start Debate – ask a question or choose a topic
  • Explore Debates – browse example topics and arguments
  • About – learn what DebAIt is and how it works

This is just the front-end. Later, we can connect it to an AI backend (Flask, Node, etc.).
